## Deprecations
- binding: Deprecate `ValidateErrFactory`, use `server.WithCustomValidatorFunc` instead (#1484)

## Features
- loadbalance: Add `LoadbalancerCtx` interface with `PickCtx(ctx, req, result)`; `BalancerFactory.GetInstance` prefers it over `Pick` when the balancer implements it, enabling context/request-aware instance selection (#1500)

## Fixes
- http1: Reject ambiguous request framing to prevent HTTP request smuggling (RFC 9112 §6.3); `Transfer-Encoding` combined with `Content-Length`, conflicting duplicate `Content-Length`, multiple `Transfer-Encoding` lines, and any `Transfer-Encoding` other than `chunked` are now rejected as malformed (#1490)
- http1: Fix panic when the request context pool is disabled; build the context via the pool's `New` instead of a bare `&app.RequestContext{}` (#1496)
- binding: Fix promoted fields of anonymous (embedded) structs not binding; JSON/form/query values now correctly override their `default` tags (#1480)

## Improvements
- network/standard: Detect client disconnection for `SenseClientDisconnection` via the `cloudwego/gopkg/net/connstate` remote-close callback, replacing the stateful-conn read-error path (#1481)
- network/standard: Rebuild connection read/write buffering on `cloudwego/gopkg/bufiox`, replacing the custom link-buffer (#1482)
- network/standard: Reduce client memory usage by avoiding references to the underlying read buffer in `Conn.ReadByte` and fixed-size body reads (#1483)
